# **App Name**: Keynique | Premium Landing

## Core Features:

- Hero Section Showcase: Displays the brand's core message 'Small Detail. Massive Personality.' with a high-quality product visual, scroll animation, and 'Shop the Collection' CTA, featuring subtle neon glow effects.
- Featured Keychains Display: Showcases a grid of 3-6 keychains with minimal product cards, implementing hover glow effects and a quick 'View Store' or 'Shop Now' button.
- Brand Value Proposition: Presents the 'Why Keynique' section detailing premium materials, unique designs, everyday accessory benefits, and collectible drops using minimal icons with a neon outline style.
- Lifestyle Imagery Gallery: Features a clean modern grid showcasing keychains in real-life contexts like on bags, car keys, and backpacks to demonstrate their aesthetic appeal.
- Conversion Call to Action: A dedicated section with the headline 'Upgrade Your Keys.' and an animated glowing neon 'Enter the Store' button to redirect users to the store link.
- Minimalist Footer: Includes the Keynique logo, essential links (Store, Instagram, Contact) separated by a subtle neon line, maintaining a clean and understated design.
- Interactive Animations & Responsiveness: Implements smooth scrolling, hover glow effects, button ripple animations, subtle parallax on the hero product, and ensures full responsiveness across all device sizes for a premium user experience.

## Style Guidelines:

- Primary interactive color: A vibrant purple (#8B5CF6) used for key call-to-action buttons, prominent text, and primary highlights, chosen for its futuristic and premium glow effect.
- Background color: A deep, almost-black tone (#0B0B0F) creating a dark minimalist foundation that emphasizes content and allows neon elements to truly stand out.
- Accent color: A striking cyan (#22D3EE) applied sparingly for complementary highlights, hover effects, and secondary interactive elements, providing a cool contrast.
- Headline font: 'Space Grotesk' (sans-serif) for its modern, techy, and futuristic feel, ideal for strong impact headlines.
- Body font: 'Inter' (sans-serif) for clear readability and a clean, neutral look, perfectly complementing the minimalist aesthetic.
- Use sleek, minimalist icons with a subtle neon outline style to align with the futuristic and clean design ethos.
- A dark mode base with a minimalist layout, ample whitespace, and clean modern grids across all sections to ensure content is uncluttered and visually impactful.
- Subtle animations like fade-ins, slide-ups, hover glow effects, a delicate button ripple effect, and slight parallax on the hero image to provide a premium and smooth user interaction without being overwhelming.
